# Scripts de Análise

Scripts Python para análise e monitoramento do sistema.

## Scripts Disponíveis

- **check_grafana_dashboard.py** - Verificação e análise de dashboards do Grafana

## Uso

```bash
# Verificar dashboard do Grafana
python check_grafana_dashboard.py
```

## Descrição

Estes scripts fornecem ferramentas de análise para monitorar e verificar o estado do sistema e seus componentes.
