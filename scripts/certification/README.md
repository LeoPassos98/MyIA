# Scripts de Certificação

Scripts para certificar modelos de IA.

## Scripts Principais

- **certify-all-via-api.sh** - Certificar todos via API (RECOMENDADO)
- **certify-all-direct.sh** - Certificar todos via script direto
- **certify-all-models-auto.sh** - Certificação automática com monitoramento

## Uso

```bash
# Certificar todos os modelos via API
./certify-all-via-api.sh
```

## Descrição

Estes scripts automatizam o processo de certificação de modelos de IA, garantindo que todos os modelos estejam validados e prontos para uso em produção.
