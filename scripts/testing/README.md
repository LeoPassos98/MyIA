# Scripts de Teste

Scripts para testar funcionalidades da aplicação.

## Categorias

### Testes de Certificação
- **test-manage-certifications.sh** - Suite completa de testes de certificação
- **test-manage-certifications-automated.sh** - Testes automatizados
- **test-manage-certifications-direct.sh** - Testes diretos
- **test-manage-certifications-practical.sh** - Testes práticos
- **test-practical-direct.sh** - Testes práticos diretos

### Testes de Validação
- **test_validations.sh** - Validações gerais do sistema
- **test_badge_system.py** - Testes do sistema de badges
- **test_login_validation.py** - Validação de login

### Testes de Grafana
- **test-grafana-detection.sh** - Detecção do Grafana
- **test-grafana-start-function.sh** - Teste de inicialização do Grafana

## Uso

```bash
# Executar suite de testes
./test-manage-certifications.sh

# Executar testes de validação
./test_validations.sh
```

## Descrição

Estes scripts fornecem uma cobertura abrangente de testes para garantir a qualidade e funcionalidade da aplicação.
